# Portfolio - Mon Portfolio Personnel

Bienvenue sur mon portfolio ! Ce projet est une vitrine de mon travail, qui présente mes compétences, projets, et ma passion pour le développement web.

## Description

Ce portfolio a été conçu pour mettre en avant mes projets, mes compétences en développement, et mes expériences professionnelles. Il est responsive et optimisé pour offrir une expérience utilisateur fluide, que ce soit sur desktop ou mobile.

## Fonctionnalités

- Présentation des projets personnels.
- Formulaire de contact pour me joindre.
- Design moderne et responsive.
- Utilisation de HTML, CSS, JavaScript et Bootstrap pour la mise en page.
- Section "Résumé" décrivant mon parcours scolaire et professionnel.

## Technologies Utilisées

- **HTML5** : Structure de la page.
- **CSS3** : Mise en page et styles.
- **JavaScript** : Interactivité et effets.
- **Bootstrap** : Framework CSS pour la mise en page responsive et les composants interactifs (grilles, boutons, formulaires, etc.).
- **Formspree** : Backend pour l'envoi des messages via le formulaire de contact.

## Aperçu

Voici un aperçu de ce que vous trouverez dans ce portfolio :

- Une section **"À propos de moi"** qui décrit mon parcours.
- Une section **"Résumé"** détaillant mon parcours scolaire et professionnel.
- Une galerie de projets avec des descriptions et des liens vers les projets en ligne.
- Un formulaire de contact pour que vous puissiez me joindre facilement.

## Section Résumé

Dans cette section de mon portfolio, vous trouverez un résumé complet de mon parcours académique et professionnel.
